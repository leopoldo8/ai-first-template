# External Integrations

## Serena MCP

- Each app/package has `.serena/project.yml`
- Provides symbolic code understanding (find_symbol, get_symbols_overview)
- Persistent memories per project for architecture context

## NX Cloud

- Remote caching for CI build artifacts
- Affected detection for incremental builds

## GitHub Actions

- CI pipeline with affected-only builds and tests
- AI docs validation on every PR

## Future Integrations (Template Users)

- PostgreSQL (via TypeORM in apps/api)
- Authentication provider (Clerk, Auth0, etc.)
- Payment processing (Stripe, etc.)
- Email service (Resend, etc.)
